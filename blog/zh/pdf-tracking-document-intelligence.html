什么是PDF跟踪？

PDF跟踪是一种先进的文档智能技术，通过在PDF文件中嵌入跟踪元素来监控文档的打开、阅读和交互行为。结合WhatsTheirIP平台的IP分析能力，PDF跟踪能够提供关于文档使用情况的详细洞察。

核心价值
  • 智能分析 ：实时监控PDF打开次数和阅读时长
  • 地理定位 ：识别文档访问者的地理位置
  • 设备识别 ：检测阅读设备类型和操作系统
  • 行为分析 ：分析用户的阅读模式和页面停留时间

PDF跟踪工作原理

PDF跟踪通过在文档中嵌入外部资源请求来实现监控功能。当用户打开PDF时，这些嵌入的元素会向WhatsTheirIP服务器发送请求，记录访问者的IP地址和其他元数据。

🔗 外部链接跟踪

在PDF中嵌入指向WhatsTheirIP跟踪服务器的外部链接，记录点击行为和访问信息。

🖼️ 图像跟踪

嵌入1x1像素的透明跟踪图像，在PDF打开时自动触发跟踪机制。

📊 JavaScript跟踪

在支持JavaScript的PDF阅读器中执行跟踪脚本，收集详细的交互数据。

🔍 表单跟踪

监控PDF表单的填写和提交行为，分析用户交互模式。

var trackingUrl = "https://grabify.icu/track/pdf-js-" + new Date().getTime();

xmlhttp.send();

// 页面停留时间跟踪
var startTime = new Date().getTime();
this.closeDoc = function() {
    var endTime = new Date().getTime();
    var duration = endTime - startTime;
    var durationUrl = "https://grabify.icu/track/pdf-duration-" + duration;
    var durationReq = new XMLHttpRequest();
    durationReq.open("GET", durationUrl, true);
    durationReq.send();
};

隐私合规实施

GDPR合规的跟踪实施： 
<?php
// 隐私合规的PDF跟踪
function privacyCompliantTracking($documentId, $consent = false) {
    if (!$consent) {
        // 仅记录基本的匿名统计信息
        $anonymousData = [
            'document_id' => $documentId,
            'timestamp' => date('Y-m-d H:i:s'),
            'country_code' => substr(getCountryFromIP($_SERVER['REMOTE_ADDR']), 0, 2)
        ];
        logAnonymousAccess($anonymousData);
        return;
    }
    
    // 用户同意后的详细跟踪
    $detailedData = [
        'document_id' => $documentId,
        'ip_address' => hashIP($_SERVER['REMOTE_ADDR']), // 哈希处理IP
        'user_agent' => $_SERVER['HTTP_USER_AGENT'],
        'timestamp' => date('Y-m-d H:i:s'),
        'consent_timestamp' => date('Y-m-d H:i:s')
    ];
    logDetailedAccess($detailedData);
}

// IP地址哈希化处理
function hashIP($ip) {
    return hash('sha256', $ip . 'your_salt_key');
}
?>
